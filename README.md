## Features

- Tanstack Start
- Tanstack Query
- TRPC
- Better Auth
- Drizzle with Postgresql
- Vercel Image dynamic Optimization (99% same as nextjs/image)
- Shadcn/ui
- Tailwind CSS
- React Email with Tailwindcss and A script to sync colors between app and emails.
- T3 Env for environment variables management.
